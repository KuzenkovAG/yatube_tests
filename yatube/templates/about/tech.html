Приложения
Функционал сайта разбит на приложения:
  • posts (отвечает за отображение, создание, изменение постов)
  • users (авторизация и регистрация пользователей)
  • about (отображение информационных страниц)
  • core (храниллище контекст процессоров, пользовательских фильтров)
Приложение posts

Публикация поста
Посты могут публиковать только авторизованные пользователи. Проверка авторизации осуществляется с помощью встроенного декоратора @login_required.
Изменение поста
Пост может изменить только пользователь, создавший пост. Проверка осуществляется с помощью созданного декоратора @owner_only.

URL:
/                     - главная страница
group/group_slug/     - страница группы постов
profile/username/     - страница пользователя
posts/post_id/        - страница поста
create/               - страница для создания поста
posts/post_id>/edit/  - страница для изменения поста
Модели:
Group - поля: id, название, slug, описание
Post  - поля: id, текст, дата публикации, группа (внешний ключ Group), автор(внешний ключ от User)
Формы:
PostForm - форма для создания/редактирования поста
Приложение Users

Приложение позволяет осуществлять:

  • Вход пользователя
  • Выход пользователя
  • Регистрацию пользователя
  • Смену пароля (для авторизоанных пользователей)
  • Сброс пароля через почту (для неавторизованных пользователей)

Приложение использует встроенные View-классы приложения django.contrib.auth. Использование которых сократило время на разработку сайта. Были переопределены только используемые <шаблоны>

Приложение используется для отображения статических информационных страниц

URL:
about/author/         - страница об авторе
about/tech/           - страница о технологии сайта
Приложение Core

Вспомогательное приложение для хранения пользовательских context processors, filter

Шаблоны

Встроенный шаблонизатор позволяет генерировать страницу на основании имеющихся шаблонов

Шаблоны страниц располагаются в папке template 
template
├── about       - шаблоны приложения about
├── includes    - общие шаблоны отображения поста, формы, паджинатора, заголовка и подвала страницы
├── posts       - шаблоны приложения posts
├── users       - шаблоны приложения users
└── base.html   - основа страницы
